# Projeto-PhP-Unipe

Boas-vindas ao README do nosso projeto!

Este site foi desenvolvido para atender às necessidades de uma barbearia ou barbeiro autônomo, oferecendo um site de maneira prática, bom visual e intuitiva. Seu objetivo principal é facilitar a rotina de atendimento e servir como uma ferramenta de divulgação profissional, ajudando o barbeiro a apresentar seus serviços de forma moderna e acessível.

O site foi pensado para funcionar como uma vitrine digital. Na página principal, o usuário encontra uma estrutura organizada em tópicos que destacam todas as informações essenciais. O header apresenta as seções principais do site e facilita a navegação. Logo abaixo, o visitante pode visualizar os valores dos cortes, encontrar exemplos de estilos de cabelo, conhecer os contatos da barbearia ou barbeiro, além de conferir a localização através dos endereços destacados. Tudo isso contribui para que qualquer cliente compreenda rapidamente os serviços oferecidos e saiba como chegar até o local.

A seção de agendamento é um dos pontos centrais do projeto. Nela, o cliente tem autonomia para marcar um horário em poucos cliques. Basta selecionar o dia desejado e escolher entre os horários disponíveis. O sistema apresenta os horários por meio de botões que visualmente indicam se o horário está livre ou ocupado, garantindo praticidade e evitando conflitos na agenda. Todas as informações ficam automaticamente registradas e organizadas diretamente no arquivo JSON do site.

Do lado do Admin, existe uma área exclusiva onde o barbeiro (ou qualquer pessoa autorizada com login e senha) pode visualizar a agenda completa do dia. É possível marcar, desmarcar ou editar horários diretamente pelo painel, sem precisar alterar manualmente o arquivo JSON que armazena os dados. Isso torna o controle muito mais seguro e simples, evitando erros e agilizando o fluxo de trabalho.

Em resumo, este projeto entrega uma solução funcional para gerenciamento de agendamentos, apresentação de serviços e organização da rotina, tornando o atendimento mais fácil tanto para o cliente quanto para o profissional.
