# Projeto-PhP-Unipe

O nosso projeto é um site de uma barbearia / barbeiro, que nele apresentamos opções de cortes de cabelo, horários disponíveis para agendar o seu corte de cabelo, preços dos tipos de corte / serviços que pode oferecer e contato junto com um portifólio do barbeiro.

Na parte de Admin do site, teremos opções de mudar os preços dos cortes, alterar os horários disponíveis, adicionar ou tirar estilos de cortes como exemplo e produtos que pode ter para oferecer.
